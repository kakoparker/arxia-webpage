Portfolio Page UX — Pick an approach

How should visitors explore the full portfolio on the dedicated /portfolio page?

A

Tabbed Domain Filter

Horizontal tab bar at the top with all domain categories. Clicking a tab filters the grid below. "All" tab shows everything. Cards animate in/out on filter change.

Pros

  • Familiar pattern, easy to scan
  • Simple implementation
  • Shows project count per domain

Cons

  • Flat — all cards look the same
  • Tab bar gets crowded with 8 domains
  • No narrative structure per domain
B

Domain Sections with Anchor Nav

Page scrolls through domain sections vertically. Each section has its own header, description, and project cards. A sticky side nav or top nav lets you jump between sections.

  • SEO-friendly with distinct sections

Cons

  • Side nav takes horizontal space on mobile
C

Domain Cards → Expandable Sections

Landing view shows 8 domain "cards" as the entry point. Clicking a domain expands inline (or scrolls to) its project list. Compact overview first, details on demand.
